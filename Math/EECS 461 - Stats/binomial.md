Binomial PMF counts the number of 1's in a block of n bits

**PMF**
$$P_D(d) = \begin{cases}  
{n\choose d} d p^d(1-p)^{n-d} & \text{if } d =0, 1,2,...,n\\  
0 & \text{else}
\end{cases}
$$
where n is a some positive number and 
p: $0 \leq p \leq 1$ 

**EXPECTED VALUE**
$E[Y] = \mu_y = n*p$
