# Textual Data

Textual data is a type of unstructured text. Textual information is organized into documents like books, PDFs, web pages, or even a text value attribute in a database. Generally information retrieval means going into unstructured data and querying it for information, information retrieval systems allow you to get documents with some kind of relevant information. This can be a set of keywords or tags.
