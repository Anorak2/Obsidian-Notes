# Using John the Ripper
https://www.openwall.com/john/

John the ripper is a free and open source password cracking tool that runs on basically everything. It supports a bunch of different attacks including dictionary attacks, hybrid attacks, and brute force attacks. It automatically detects the type of password, and there are in fact distributed versions also for if you are serious.
