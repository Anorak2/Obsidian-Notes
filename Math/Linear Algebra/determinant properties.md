to calculate determinants see [[Quick way to find determinant]] 

if A is a square, invertible matrix and $|A| \space!= 0$ then $|A^T| = |A|$, and if |A| != 0 then
$$|A^{-1}| = \frac{1}{|A|}$$
### Other:
$$ A = 
\begin{bmatrix}
a & b \\
c & d \\
\end{bmatrix}$$
here we are using the [[Adjoint]] of A
$$A^{-1} = \frac{1}{|A|}adj(A)$$
$$A(adj A) = |A|I_n$$

### Determinant of an Elementary Matrix
if we added a constant multiple of one row to another row then $|E| = 1$

if we multiplied a row by a scalar c then $|E| = c|I_n| =c$

if we switched two rows $|E| = -1|I_n| = -1$
