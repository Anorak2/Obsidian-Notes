# Data Life Cycle

**Data Generation**
This is all of the posts or uploads or books that keep getting created and updated, this happens just all the time.

**Data Collection**
Data can be collected in a ton of ways: forms, transactions, observation, research, and many more

**Data Processing**
Once we have it we have to process it. Data wrangling is turning data into a useful form, data compression which is transforming data into a more efficient format, and data encryption.

**Data Storage**
Data has to be stored for it to be useful

**Data Management**
Includes modeling, organizing, storing, querying data over the life of the project.

**Data Analysis**
the process of generating meaningful insights from raw data

**Visualization**
Creating graphical representation representations of data

**Interpretation**
an opportunity for making better sense of data analysis and visualization
