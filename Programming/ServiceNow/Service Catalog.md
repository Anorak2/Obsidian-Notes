
[[ServiceNow]]

The service Catalog is a way for ordering various services and products offered by various departments for users. It organizes items and puts it all in one place for users.

**Components**
We fill our store with items. These can be edited in **All > Service Catalog > Catalog Definitions > Maintain Items**. These variables are global by default and define the questions for the end user ordering the catalog items. There are also **variable sets** which allows for variables to be shared across some groups.

Record producers are kind of like simplified forms that allow the users to provide information that gets translated into the database. We can use this to have a simplified menu for reporting a defect that gets translated into an HR case.
