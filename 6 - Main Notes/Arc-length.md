# Arc-length
Arc length is the length of a curve in between two points when the curve is smooth.

The length of a given line segment is:
$$ \sqrt{(x')^2+(y')^2}$$
which after a number of transformations turns into
$$\text{Arc Length}=\int_a^b \sqrt{1+[f'(x)]^2 }\space dx$$
