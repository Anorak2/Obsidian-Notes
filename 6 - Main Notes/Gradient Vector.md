# Gradient Vector
The gradient vector $\nabla f$ is equal to the partial derivative for each variable 

$$\nabla f= \langle f_x(x,y,z), \space f_y(x,y,z), \space f_z(x,y,z) \rangle$$
the gradient vector points in the direction of the greatest increase, and it is orthogonal to the tangent line of $f$ at (a,b)
