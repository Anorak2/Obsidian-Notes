# Big Data
**Volume**
This is the size of data managed by the system, for big data the data ranges from Terabytes to Petabytes to Exabytes.

**Velocity**
The type and frequency of the data, as well as the speed of data creation, ingestion, and processing. This also includes the speed with which data comes and goes.

**Variety**
The type of data and source, also the range of data types and source that are used.

**Veracity**
- Credibility of the source
- Suitability of data for the target audience

**Value**
- TCO and the ROI (Return on investment) for the data
